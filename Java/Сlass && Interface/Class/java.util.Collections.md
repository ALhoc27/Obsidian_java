- **Утилитный класс:** Все методы `Collections` являются статическими, поэтому объект этого класса не создается.
- Поддерживает работу с интерфейсами `Collection`, `List`, `Set`, `Map` и их наследниками.

##### Методы сортировки:
- [`static <T> void sort(List<T> list)`](https://docs.oracle.com/en/java/javase/21/docs/api/java.base/java/util/Collections.html#sort(java.util.List)) [^1]: Сортирует элементы списка в естественном порядке.
