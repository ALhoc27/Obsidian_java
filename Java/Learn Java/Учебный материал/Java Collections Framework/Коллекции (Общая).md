В Java **Java Collection Framework** представляют собой группу интерфейсов и классов(их иерархию), предназначенных для удобного и эффективного управления и хранения наборов **однородных** объектов. Коллекции предоставляют различные структуры данных и алгоритмы для работы с ними, позволяя разработчику пользоваться большим количеством структур данных из «коробки».
