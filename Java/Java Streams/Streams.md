
```java
String[] array = {"apple", "banana", "cherry"};
ArrayList<String> list = Arrays.stream(array)
                               .collect(Collectors.toCollection(ArrayList::new));

```
Метод `Arrays.stream(array)` принимает массив `array` и преобразует его в `Stream`[^1]

`.collect(Collectors.toCollection(ArrayList::new))` 
После создания `Stream` от массива, мы вызываем `.collect(...)`, который собирает элементы потока в коллекцию.
- **`.collect(...)`** — это терминальная операция потока, которая преобразует `Stream` в другую форму, обычно в коллекцию. В данном случае, она собирает все элементы в `ArrayList`.
- **`Collectors.toCollection(ArrayList::new)`** — это специальный коллектор, который указывает, что каждый элемент потока должен быть собран в новую коллекцию типа `ArrayList`.



[^1]:`Stream` — это последовательность элементов, которые можно обрабатывать по одному. В данном случае, `Stream` создается для элементов массива типа `String`
	
	- **Что делает `Stream`?** Потоки в Java позволяют обрабатывать данные в функциональном стиле. С помощью них можно выполнять фильтрацию, сортировку, преобразования и другие операции с данными.
	- **Почему это полезно?** Потоки помогают выразить сложные манипуляции над данными лаконично и эффективно, особенно при работе с большими коллекциями данных.
