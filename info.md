В Obsidian есть несколько встроенных типов блоков, которые можно использовать для выделения информации в заметках. Вот основные типы:

1. **[!NOTE]** – для обычных заметок.
2. **[!TIP]** – для полезных советов.
3. **[!INFO]** – для предоставления дополнительной информации.
4. **[!WARNING]** – для предупреждений.
5. **[!DANGER]** – для обозначения опасных действий или серьезных предупреждений.
