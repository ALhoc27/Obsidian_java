**`IntentService`** — базовый класс для служб, работающих в фоновом режиме.
    
    - Выполняет задачи последовательно на отдельном потоке.
    - После завершения всех задач автоматически завершает работу.

**Перейдем к `MainService`.**  
Этот класс чаще всего организует логику других служб. У него есть несколько методов:

- `startService`: запускает службу.
- `stopService`: завершает службу.
- `statusService`: проверяет состояние и готовность службы.
