- Вызывается при завершении сервиса, как с помощью `stopService()`, так и при завершении самим сервисом через `stopSelf()`.
- Используется для освобождения всех ресурсов и завершения задач, запущенных в сервисе.
```java
@Override
public void onDestroy() {
    super.onDestroy();
    // Освобождение ресурсов и завершение работы
}
```
